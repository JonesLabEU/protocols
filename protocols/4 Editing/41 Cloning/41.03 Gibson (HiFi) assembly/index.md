# Gibson (HiFi) assembly

## Procedure

1. Set up the following reaction:

| Component                              | Amount   |
| -------------------------------------- | -------- |
| Empty vector                           | .1 pmol  |
| Insert                                 | .2 pmol  |
| NEBuilder HiFi DNA Assembly Master Mix | 10 uL    |
| Milli-Q                                | to 20 uL |

2. Incubate samples in a thermocycler at 50C for 60 min, then store at +4C.
	- Could be shortened to 15 min.
	- For a long-term storage, place samples to -20C.
3. Transform with 2 uL (10%) of the **chilled** assembled product.
