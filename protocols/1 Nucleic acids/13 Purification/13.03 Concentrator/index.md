# SpeedVac concentrator
[Manual](https://assets.thermofisher.com/TFS-Assets%2FLED%2Fmanuals%2FDNA130 User Manual final versionpdf.pdf)

1. Turn on the device.
2. Pull the lid open with your hands.
3. Add your samples (in Eppendorfs) and counterbalancing weights.
4. Choose a 35C heating temperature (the default when you turn on the device).
5. Set run time.
6. Run using "Auto Run". It will stop automatically after the set time and start beeping. ("Manual Run" doesn't stop even if run time was set.)
7. Wait for the rotor to stop. Then the lid will unlock.
